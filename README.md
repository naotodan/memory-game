# Memory Game

数字を覚えて、順番通りにタップする記憶力トレーニングゲームです。

## 遊び方

1. 画面に表示された数字を10秒間覚える
2. カードが裏返ったら、数字を順番通りにタップ
3. 全てタップできればクリア！

## レベル

| レベル | 特徴 |
|--------|------|
| Lv 1 | 数字を1から順番にタップ |
| Lv 2 | ランダムな順番でタップ |
| Lv 3 | カードが動く！さらに難しい順番で |

## 特徴

- コンボボーナスで高スコアを狙え
- ライフ制でミスしても続けられる
- ヒント機能で全カードを一度だけ確認可能
- ベストスコアを記録してランク評価（S〜D）
